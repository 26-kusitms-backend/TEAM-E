## 🙋‍♀️ 스터디 진행 방식

- 시간 : **매주 일요일 12:30**
- 장소 : **충무로 이리로 카페**
- 공부 방식 : **책 한권을 정해 해당 책의 Chapter별 담당 조원을 정해 해당 조원이 공부한 내용을 정리하며, 관련 내용을 나누는 방식 / 참고 자료 : 자바 웹 개발 워크북 (구멍가게 코딩단 / 프리렉)**
- 설명 방식 : **Chaper별 정리되어 있는 목차 순서대로 내용 정리 / 실습 코드 리뷰**
- 기록 방식 : **본 레포지토리에 PR로 기록**
- 인증 방식 : **PR 리뷰**
